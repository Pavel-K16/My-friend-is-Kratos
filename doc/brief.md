# Общие сведения

## Персонажи картины

Главный герой картины — **Кратос**, персонаж из игры "Бог Войны". По сюжету игры, Кратос носит на своём поясе отрубленную голову **Мимира** — живого персонажа, который периодически разговаривает с Кратосом. Голова Мимира является вторым действующим персонажем на картине.

## 1. Формат картины

Соотношение сторон картины предположительно будет **4:5** или **9:16** (вертикальный/портретный формат).

## 2. Ракурс

Два главных действующих персонажа (Кратос и голова Мимира) на картине расположены в анфас. Используется только такой ракурс.

## 3. Оружие

В правой руке Кратоса должен быть топор Левиафан, а в левой руке — огненный клинок.

## 4. Голова Мимира на поясе

Голова Мимира (второго персонажа в этой концепции) будет расположена на поясе Кратоса и жёстко закреплена там верёвками.

## 5. Тип локации и задний фон

Тип локации и задний фон обсуждаются — зависят от выбранного формата картины и соотношения сторон. Возможные варианты фонов представлены в референсах.

## 6. Настроение картины

Настроение картины полностью передаётся исходным референсом (лицо друга).

## 7. Дополнительные варианты

Дизайнерские решения и предложения приветствуются. Я всегда открыт и готов к обсуждению.

## Структура референсов

```
References/
├── kratos-refs/                    # Основные референсы персонажа Кратоса
│   ├── other-kratos-refs/          # Дополнительные референсы Кратоса
│   └── settings-refs/              # Референсы окружения и фона
├── kratos-face-refs/                # Референсы лица и мимики Кратоса
├── real-friends-faces/             # Исходные фотографии лица друга
├── blades-refs/                    # Референсы клинков Хаоса
├── axes-refs/                      # Референсы топора Левиафан
└── mimirs-refs/                    # Референсы головы Мимира
```

### Описание уровней вложенности

**Корневой уровень (`References/`):**
- Содержит все категории референсов, организованные по типам элементов картины

**Первый уровень вложенности:**
- `kratos-refs/` — основные материалы персонажа Кратоса
- `kratos-face-refs/` — детальные референсы лица
- `real-friends-faces/` — исходные фотографии для интеграции
- `blades-refs/` — материалы по оружию (клинки)
- `axes-refs/` — материалы по оружию (топор)
- `mimirs-refs/` — материалы по голове Мимира

**Второй уровень вложенности (внутри `kratos-refs/`):**
- `other-kratos-refs/` — дополнительные референсы персонажа с различными позами и ракурсами
- `settings-refs/` — референсы локаций и окружения для фона картины
