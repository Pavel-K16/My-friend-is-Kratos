# Структура референсов для проекта "Мой друг - Кратос"

## Общая структура папок с референсами

### 1. Kratos Refs/ (Основные референсы Кратоса)
**Назначение:** Основные визуальные материалы персонажа Кратоса из игры "Бог Войны"

**Содержание:**
- `Главный референс Кратоса.png` - основной референс для понимания общего образа
- `Other Kratos Refs/` - дополнительные референсы персонажа
  - Референсы с различными позами и ракурсами
  - Референсы с оружием (топор, клинки)
  - Референсы эмоций и выражений лица
- `Settings Refs/` - референсы окружения и фона
  - Референсы локаций из игры
  - Референсы атмосферы и настроения сцены

### 2. Kratos Face Refs/ (Референсы лица Кратоса)
**Назначение:** Детальные референсы лица, мимики и черт лица персонажа

**Содержание:**
- Референсы бороды и растительности на лице
- Референсы выражения лица (гнев, решимость, концентрация)
- Референсы ракурсов лица (анфас, профиль, три четверти)

### 3. Real Friends Faces/ (Фотографии лица друга)
**Назначение:** Исходные фотографии лица друга для интеграции в образ Кратоса

**Содержание:**
- `Для лица Кратоса.jpeg` - основная фотография для замены лица
- `Для лица Мимира.jpeg` - дополнительная фотография (если планируется интеграция с Мимиром)

**Требования к фотографиям:**
- Высокое разрешение
- Хорошее освещение
- Различные ракурсы (желательно)
- Естественное выражение лица

### 4. Blades Refs/ (Референсы клинков Хаоса)
**Назначение:** Референсы оружия Кратоса - клинков Хаоса

**Содержание:**
- Основной референс клинков
- Дополнительные референсы с разных ракурсов
- Референсы клинков в действии (с огненными эффектами)

### 5. Axes Refs/ (Референсы топора Левиафан)
**Назначение:** Референсы топора Левиафан - основного оружия Кратоса

**Содержание:**
- Основной референс топора
- Дополнительные референсы с деталями
- Референсы топора в руках персонажа

### 6. Mimirs Refs/ (Референсы головы Мимира)
**Назначение:** Референсы для возможной интеграции образа друга в голову Мимира

**Содержание:**
- Референс головы Мимира
- Референс закрепления головы Мимира на поясе Кратоса
- Референсы для понимания масштаба и пропорций

## Принципы организации референсов

### По приоритету использования:
1. **Критически важные** - используются в первую очередь
   - Главный референс Кратоса
   - Фотография лица друга
   - Референсы окружения

2. **Важные** - используются для детализации
   - Референсы оружия
   - Референсы мимики и выражений
   - Референсы поз и ракурсов

3. **Дополнительные** - используются для вдохновения и уточнения деталей
   - Альтернативные референсы
   - Референсы атмосферы
   - Референсы для Мимира (если применимо)

### По категориям:
- **Персонаж** - все, что связано с внешним видом Кратоса
- **Оружие** - клинки и топор
- **Окружение** - локации и фон
- **Интеграция** - материалы для объединения лица друга с телом Кратоса

## Рекомендации по использованию

1. **Начинать с главных референсов** - сначала изучить основной образ Кратоса
2. **Сопоставлять пропорции** - сравнить фотографию друга с референсами лица Кратоса
3. **Учитывать освещение** - подобрать референсы с похожим освещением
4. **Соблюдать стиль игры** - все референсы должны соответствовать визуальному стилю "Бог Войны"
